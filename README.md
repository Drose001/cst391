The Grand Canyon University Introduction to JavaScript Web Application Development (CST-391) course focuses on modern full-stack development practices. Throughout this course, students explored both frontend and backend technologies, including React, Node.js, Express, Angular and MySQL. The course emphasized building scalable web applications and integrating multiple technologies into a complete system.

---

## Introduction

Throughout this course, I developed a strong understanding of full-stack web application development. I worked on multiple activities and milestone assignments that focused on building, connecting, and refining a complete web application.

Key areas of learning included:
- Frontend development using React
- Backend development using Node.js and Express
- Database design and integration using MySQL
- REST API development and testing
- Debugging and troubleshooting real-world issues

This course provided hands-on experience in building a fully functional application from the ground up, preparing me for real-world software development.

---

## Presentation Artifacts

The primary artifact for this course is the **SmartCart Grocery Planner** application.

Artifacts include:
- Full-stack application (React + Node.js + MySQL)
- REST API endpoints and testing (Postman)
- User interface screenshots and demonstrations
- README documentation and markdown formatting
- Screencast demonstration

These artifacts demonstrate the ability to design, build, and document a complete web application using modern technologies.

## Conclusion

The SmartCart Grocery Planner project represents a complete full-stack application developed using modern web technologies. It demonstrates the integration of a React frontend, an Express-based backend, and a MySQL database to deliver a functional and user-focused system.

This project builds upon the foundational concepts developed throughout the course activities, particularly the Music API application, where I first implemented RESTful services, database integration, and backend architecture. These early exercises provided the groundwork that was later expanded into a full-stack application in SmartCart.

Throughout this course, I gained practical experience in:

Integrating frontend and backend systems to enable seamless data flow
Designing and managing relational databases for structured data storage
Developing and testing RESTful APIs to support application functionality
Troubleshooting real-world development challenges, including connectivity and configuration issues
Creating professional documentation using Markdown

In particular, developing the README.md using Markdown proved to be a valuable skill. It allowed me to clearly document the application, communicate technical concepts effectively, and present the project in a structured and professional format. This experience highlighted the importance of documentation in real-world development environments, where it supports collaboration, onboarding, and long-term maintainability.
